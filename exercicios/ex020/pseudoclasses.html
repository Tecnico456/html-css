"display" é um parametro para colocar o tipo de exibição; Junto com o "inline-block" que é um tipo de exibição que permite que os elementos se comportem como blocos, mas ainda se alinhem horizontalmente

"height" é um parametro para colocar a altura

"width" é um parametro para colocar a largura

"line-height" é um parametro para colocar a altura da linha

"margin" é um parametro para colocar a margem

"border" é um parametro para colocar a borda onde: "1px" e a largura da borda; "solid": É o tipo de borda ; "black" é a cor da borda

"text-align" é um parametro para alinhar o texto

"background-color" é um parametro para colocar a cor de fundo

"lightgray" é uma cor de fundo clara

"black" é a cor da borda

"#" é um "id"; "." é uma 'classe' ; ":" é uma pseudo-classe.

As pseudo-classes são usadas para definir o estado de um elemento, como quando ele é clicado, focado ou quando o mouse passa sobre ele.

pseudo-classe esta relacionado ao estado de um elemento ex: se ele está ativo, se ele está marcado, se ele está vazio, se ele está habilitado,

"hover" é um pseudo-classe, é um pseudo-classe de vários elementos, no caso deste documento esta sendo aplicado ao elemento "div"

"hover" significa que: quando eu passar o mouse por cima


        <style>
            div:hover {
                background-color: yellow;
                color: red;
            }
        </style>

O código acima muda a cor de fundo do elemento "div" para amarelo e a cor do texto para vermelho quando o mouse passa sobre ele.

"hover" faz uma configuração de estado; /como no exemplo acima citado, ele sáo vai ficar amarelo quando eu passar o mouse sobre ela, que é o "hover" significa que: quando eu passar o mouse por cima
