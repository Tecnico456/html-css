Capítulo 1.1

Os Métodos de Estilo do CSS Inline, são metodos de mudança pontual, pois, são mudanças que são feitas linhas por linhas; paragrafo por paragrafo, titulo por titulo; etc.
O que torna o CSS Inline, um método de estilo pontual, é o fato de que ele não altera o estilo de outros elementos, apenas do elemento que está sendo modificado.
O CSS Inline é utilizado para fazer mudanças pontuais em um elemento específico, sem afetar os outros elementos da página.

Capítulo 1.2

O Estilo Inline significa o seguinte: Que o estilo é feito na mesma linha que o CSS, e todos os Estilos e todas as configurações são feitas na mesma linha
