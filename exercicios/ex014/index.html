Método de Estilos Locais / Internos

Capítulo 1.1

Nos Método de Estilos Locais / Internos, você aplica os estilos CSS tudo dem uma vez na parte de cima do código logo abaixo da tag "title", assim voce pode fazer a modificação da pagina toda de uma só vez sem ter que ficar colocando css em cada tag uma por uma.

Capítulo 1.2

Existem dois provlemas nesta tecnica:

  • O primeiro é que o código CSS fica misturado com o HTML, o que pode dificultar a leitura e a manutenção do código.
  • O segundo é que, se você tiver aplicar os mesmos estilos em várias páginas que contituem um site, terá que repetir o código CSS em cada uma delas.

Capítulo 2

Capítulo 2.1

Este tipo de tecnica CSS é últil quando vai fazer a modificação em um documento apenas.

Capítulo 2.2

Uma vantagem adicional do Método de Estilos Locais / Internos é que ele permite testar rapidamente alterações de estilo em uma única página, sem a necessidade de criar ou modificar arquivos externos. Isso pode ser útil durante o desenvolvimento inicial ou para prototipagem rápida.
