Exemplo de caixas Box-level

Pparágrafos também são exemplos de box-level, mas os Links são exemplos de caixas inline-level. Vamos ver como tudo isso finciona.

Margin fica da Borda para fora; o Padding fica da borda para dentro

Para configurar as Box como Margin e Padding, configura-se no sentido Horário

Exemplo

  • 10px para o topo
  • 20px para a direita
  • 30px para a base
  • 40px para a esquerda

Para centralizar a box você coloca a Margin em auto

Exemplo

  • Margin Top: 10px
  • Margin Right: auto
  • Margin Bottom: 10px
  • Margin Left: auto

para simplificar pode colocar somente:

  • Margin: auto;

    • Para simplificar o código pode-se usar o shorthands

    Exemplo:

    • Padding: 10px 20px 30px 40px;
      • //Padding no sentido horário
    • Padding: 10px 20px;
      • //Padding 10px para o topo e base, e 20px para a direita e esquerda
    • Padding: 10px;
      • //Padding uniforme de 10px em todos os lados
    • Border: 10px solid blue;
      • // Borda de 10px, estilo sólido e cor azul
    • Border: 10px solid;
      • // Borda de 10px, estilo sólido, cor padrão
    • Border: 10px;
      • // Borda de 10px em todos os lados
    • Margin: 20px;
      • // Margem de 20px em todos os lados
    • Margin: 20px 10px;
      • // Margem de 20px para o topo e base, 10px para a direita e esquerda
    • Margin: 20px 10px 30px;
      • // Margem de 20px para o topo, 10px para a direita e esquerda, 30px para a base
    • Margin: 20px 10px 30px 40px;
      • // Margem de 20px para o topo, 10px para a direita, 30px para a base, 40px para a esquerda
    • Outline: 5px solid blue;
      • // Contorno de 5px, estilo sólido e cor azul
    • Outline: 5px solid;
      • // Contorno de 5px, estilo sólido, cor padrão
    • Outline: 5px;
      • // Contorno de 5px
    • Outline: 5px dashed salmon;
      • // Contorno de 5px, estilo tracejado e cor salmão
