Gruping Tags

As tags de agrupamento sempre foram 2

O div é um agrupamento de blocos, ou seja, de caixas de nível de bloco

O span é um agrupamento de caixas de nível de linha, ou seja, de caixas inline-level

Existem Gruping tags semânticas, como header, footer e article.

Essas tags semânticas são usadas para dar significado ao conteúdo, facilitando a compreensão e a acessibilidade.

Por exemplo, a tag header é usada para agrupar o cabeçalho de uma página faz parte do conteúdo do seu site, enquanto a tag footer é usada para agrupar o rodapé.

A tag main é usada para agrupar o conteúdo principal de uma página.

Exemplo:

Bem-vindo ao meu site

  • Início
  • Sobre
  • Contato

Com a tag nav você pode agrupar links de navegação.

Este é o conteúdo principal da página.

Fazendo desta forma o site com Gruping Tags você ja tem a estrutura do seu site modelado, e voce consegue com CSS estilizar cada parte do seu site de forma independente. Deixando assim mais fácil de aplicar a CSS em todo o seu site e deixando ele cada vez mais bonito.

box-shadow

A propriedade box-shadow é usada para adicionar sombras a elementos. Ela aceita valores para deslocamento horizontal, deslocamento vertical, desfoque, espalhamento e cor.

Exemplo de uso da propriedade box-shadow:

                        
                            box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);

Isso cria uma sombra com deslocamento de 2 pixels para a direita e para baixo, um desfoque de 5 pixels e uma cor preta com 30% de opacidade.

Você pode usar a propriedade box-shadow para adicionar sombras a elementos como caixas, botões e imagens.

Exemplo de uso da propriedade box-shadow:

                        
                            .caixa {
                                width: 200px;
                                height: 100px;
                                background-color: lightblue;
                                box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
                            }

Isso cria uma caixa com largura de 200 pixels, altura de 100 pixels, cor de fundo azul claro e uma sombra com deslocamento de 2 pixels para a direita e para baixo, um desfoque de 5 pixels e uma cor preta com 30% de opacidade.

Você pode usar a propriedade box-shadow para adicionar sombras a elementos como caixas, botões e imagens.

Exemplo de uso da propriedade box-shadow:

                        
                            .botao {
                                padding: 10px 20px;
                                background-color: blue;
                                color: white;
                                border: none;
                                border-radius: 5px;
                                box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
                            }

Isso cria uma caixa com largura de 200 pixels, altura de 100 pixels, cor de fundo azul claro e uma sombra com deslocamento de 2 pixels para a direita e para baixo, um desfoque de 5 pixels e uma cor preta com 30% de opacidade.

Você pode usar a propriedade box-shadow para adicionar sombras a elementos como caixas, botões e imagens.

Exemplo de uso da propriedade box-shadow:

                                padding: 10px 20px;

                                color: white;

Isso cria um botão com preenchimento de 10 pixels na parte superior e inferior e 20 pixels nas laterais, cor de fundo azul, texto branco, sem borda, bordas arredondadas de 5 pixels e uma sombra com deslocamento de 2 pixels para a direita e para baixo, um desfoque de 5 pixels e uma cor preta com 30% de opacidade.

A tag aside é usada para agrupar conteúdo relacionado, mas que não faz parte do conteúdo principal da página.

Por exemplo, você pode usar a tag aside para agrupar uma barra lateral com links relacionados, informações adicionais ou anúncios.É um conteúdo periférico ao artigo, relativo ao artigo.

exemplo de uso da tag aside:

Barra Lateral

  • Link 2
  • Link 3

A tag section é usada para agrupar conteúdo temático dentro de uma página.

Você pode usar a tag section para agrupar conteúdo relacionado, como notícias, artigos ou eventos.

Você pode usar a tag article para agrupar conteúdo independente, como artigos de blog ou notícias.

Você pode usar a tag footer para agrupar o rodapé de uma página.

O rodapé geralmente contém informações de copyright, links de contato e outras informações relevantes.

Você pode usar a propriedade border-radius para adicionar bordas arredondadas a elementos como caixas, botões e imagens.

Exemplo de uso da propriedade border-radius:

                
                    .caixa {
                        width: 200px;
                        height: 100px;
                        background-color: lightblue;
                        border-radius: 10px 20px 30px 40px;
                    }

Isso cria uma caixa com largura de 200 pixels, altura de 100 pixels, cor de fundo azul claro e bordas arredondadas com raios de 10 pixels, 20 pixels, 30 pixels e 40 pixels nos cantos superior esquerdo, superior direito, inferior direito e inferior esquerdo, respectivamente.
