Capítulo 2.1

Pode-se tambem misturar os métodos. pode colocar tambem os métodods de

estilo internos e inline, mas o mais recomendado é usar o método de estilo externo.

O método de estilo interno é usado para aplicar estilos a um único documento HTML, enquanto o método de estilo inline é usado para aplicar estilos a um único elemento HTML.

O método de estilo externo é o mais recomendado, pois permite que você aplique estilos a vários documentos HTML de uma só vez, tornando a manutenção do código mais fácil e eficiente.
